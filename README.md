# Factorio-Imitation

## 사용한 기술
* Design Pattern
    * 팩토리 메소드 (Factory Method)
    * 싱글톤 (Singleton)

* OpenGL  
    * 셰이더 (GLSL)
        * Vertex Buffer Object
        * Uniform Buffer Object
    * 인스턴싱 (Instancing)
